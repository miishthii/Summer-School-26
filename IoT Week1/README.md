# IoT Week 1

## Overview

This folder contains the Week 1 assignments completed during the **IoT Summer School 2026**. The exercises focus on learning Arduino programming fundamentals, digital input/output, arrays, LED control, push buttons, and basic embedded system concepts.

---

## Projects Included

### LED Blink

* Basic Arduino program to blink an LED.
* Demonstrates digital output and timing using `delay()`.

### Pushbutton LED

* Controls an LED using a push button.
* Demonstrates digital input and button state detection.

### LED Fade Array

* Fades multiple LEDs using PWM and arrays.
* Demonstrates analog output and array-based programming.

### LED 4×4 Matrix

* Controls a 4×4 LED matrix using row and column multiplexing.
* Demonstrates matrix scanning and efficient pin usage.

### LED 4×4 Array

* Controls LEDs using array indexing and loops.
* Demonstrates structured programming with arrays.

### Seven Segment Dice

* Simulates a digital dice using a seven-segment display.
* Displays random numbers from 1 to 6.

### Reaction Time Game

* Measures user reaction time using LEDs and a push button.
* Demonstrates timing, random delays, and user interaction.

### Additional Assignment

* Contains supplementary Arduino exercises and practice programs completed during Week 1.

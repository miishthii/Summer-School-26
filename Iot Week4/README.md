# IoT Week 4

## Overview

This folder contains the Week 4 assignments completed during the **IoT Summer School 2026**. The projects introduce wireless communication technologies using Bluetooth, Bluetooth Low Energy (BLE), Wi-Fi, and MQTT for Internet of Things (IoT) applications.

---

## Projects Included

### Bluetooth LED Controller

* Controls multiple LEDs using Bluetooth commands received through an HC-05 Bluetooth module.
* Supports turning LEDs ON/OFF and flashing LEDs remotely.

### BLE Sensor

* Uses an ESP32 to broadcast temperature and humidity readings over Bluetooth Low Energy (BLE).
* Sensor data can be viewed using BLE scanner applications such as nRF Connect or LightBlue.

### Wi-Fi Web Server

* ESP32-based web server displaying live temperature, humidity, device uptime, and LED control through a web interface.
* Automatically refreshes sensor readings every 10 seconds.

### MQTT Publisher

* Publishes temperature and humidity readings from an ESP32 to a public MQTT broker.
* Uses JSON-formatted messages and subscribes to an LED control topic for remote device control.
